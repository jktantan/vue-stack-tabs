# Installation

::: warning
Ensure Node.js >= 18. Older versions may cause errors.
:::

## Package Manager

We recommend using **pnpm**:

::: code-group

```bash [pnpm]
pnpm add vue-stack-tabs
```

```bash [npm]
npm install vue-stack-tabs
```

```bash [yarn]
yarn add vue-stack-tabs
```

:::

## Build from Source

To use the latest dev version, clone from GitHub or Gitee and build locally:

::: code-group

```bash [GitHub]
git clone https://github.com/jktantan/vue-stack-tabs.git
cd vue-stack-tabs
pnpm install
pnpm run lib:build
```

```bash [Gitee]
git clone https://gitee.com/jktantan/vue-stack-tabs.git
cd vue-stack-tabs
pnpm install
pnpm run lib:build
```

:::
