## 演示功能

使用上方按钮体验各种功能：

### 基础用法
- 🏠 **首页**：不可关闭的默认标签
- 📄 **关于/联系**：可关闭标签
- 📑 **批量打开**：一次性打开多个标签

### 高级功能
- 📜 **滚动位置**：自动保存和恢复滚动位置
- 📝 **表单状态**：切换标签后输入内容保持
- 🔗 **嵌套路由**：标签内前进/后退导航
- 🎲 **动态标签**：运行时创建唯一 ID 标签

### 标签操作
- 右键点击标签打开上下文菜单
- 关闭左侧/右侧/其他/全部
- 刷新当前或全部标签

## 试试看

1. 点击任意按钮打开标签
2. 在"表单演示"标签中填写表单
3. 切换到其他标签再回来——你的输入被保留了！
4. 在"滚动演示"标签中向下滚动，切换标签后返回——滚动位置被记住了！
5. 右键点击任意标签查看上下文菜单选项
