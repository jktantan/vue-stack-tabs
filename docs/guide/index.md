# Introduction

Vue Stack Tabs is a route-based tab component for Vue 3 and Vue Router, designed for multi-tab page management.

## Features

- [x] Open or switch tabs in response to route changes
- [x] Mouse wheel scroll when too many tabs
- [x] Tab operations: open, switch, close, refresh, reset — see [Tab Operations](../base/tab.md)
- [x] Embed external sites via iframe tabs
- [x] Customization: [transitions](../advance/transition.md), right-click context menu
- [x] [i18n](../advance/language.md)
- [x] Stack-based caching
- [x] [Initial tabs](../advance/initial.md) — default tabs on first load
- [x] Tab recovery on browser refresh
- [x] [Nuxt support](../base/nuxt.md)
- [x] [Scroll position](../advance/scroll.md) restoration when switching tabs
